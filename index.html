使用方法

4步把项目用起来

先判断价值，再去 GitHub 下载、安装、试跑。

01

选分类

按开发、设计、产品、测试安全快速筛选适合你的项目。

02

看价值

点击卡片查看项目作用、核心价值和适合解决的问题。

03

下载内容

进 GitHub 后优先看 README、Releases、examples；可下载源码 ZIP、安装包、Demo、配置文件。

04

直接试用

点 Code 下载 ZIP 或复制 git clone；按 README 执行 install/start，先跑 Demo 再接入项目。

GitHub 下载选择
源码 ZIP想快速看代码、改模板、离线研究时下载；点 Code → Download ZIP，解压后看 README。
git clone要长期跟进更新、二次开发或提交改动时使用；复制仓库地址后在终端执行 clone。
Releases 安装包想直接使用工具时优先下载；选择 Windows/macOS/Linux 对应安装包或可执行文件。
examples / Demo不确定项目怎么用时先跑示例；按 README 的 install/start 命令安装依赖并启动。
